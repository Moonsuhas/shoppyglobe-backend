# ShoppyGlobe Backend

This is the backend for the **ShoppyGlobe E-commerce application** built using **Node.js, Express, MongoDB, and JWT authentication**.

---

## Features

- User Registration and Login (JWT authentication)  
- Product Management (GET products, GET single product)  
- Shopping Cart (Add, Update, Delete items)  
- MongoDB integration with Mongoose  
- API Error Handling and Validation  

---

## Technologies Used

- Node.js  
- Express.js  
- MongoDB Atlas  
- Mongoose  
- JSON Web Tokens (JWT)  
- Nodemon (for development)
